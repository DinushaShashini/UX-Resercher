About the Role

As a Software Engineering Intern at Google, you will join one of our core product teams and work on real, production-grade infrastructure that serves billions of users worldwide. You'll be embedded with a full-time engineering team, participate in design reviews, and ship code that directly impacts Google's products.

This is a 12-week Summer internship. You will be paired with a dedicated host engineer and a peer intern buddy to ensure you have the mentorship needed to succeed.

Responsibilities

  • Design, develop, and test software components for large-scale distributed systems
  • Collaborate with engineers, PMs, and UX designers in a fast-paced agile environment
  • Participate in code reviews and contribute to engineering best practices
  • Write clean, documented, and testable code following Google's engineering standards
  • Debug production issues and contribute to incident response processes
  • Present your internship project to a cross-functional audience at the end of the program
  • Explore open-ended technical problems and propose innovative solutions

Preferred Qualifications

  • Experience with distributed systems, cloud infrastructure (GCP, AWS, Azure)
  • Contributions to open-source projects or personal portfolio on GitHub
